# 湖南出版中心 H5 交互原型

湖南出版中心（Hunan Publishing Center）交互式 H5 UI 原型，覆盖课程报名、发展历程、课程资料、效果评估等核心功能模块。

## 功能页面

- **课程报名**：活动列表 + 报名详情表单（含学段学科二级联动）
- **发展历程**：1980-2026 纵向时间轴，支持展开收起
- **课程资料**：学段→学科→培训方向三级联动筛选
- **常用功能**：效果评估、教材调换、意见反馈

## 技术栈

纯 HTML5 + CSS3 + 原生 JavaScript，零依赖。

## 预览

直接用浏览器打开 `outputs/hunan-pub-h5.html` 即可。
